🔒 Digital Inheritance Vault
🌟 Project Overview
This is a "Dead Man's Switch" security application developed in Python. It protects digital assets by encrypting them and automatically delivering them to a designated heir if the owner becomes inactive for a specified period.

🚀 Technical Features
AES-256 Encryption: Implemented using the cryptography library to ensure data privacy.

Activity Monitoring: A "Heartbeat" system that tracks user check-ins to monitor status.

Automated SMTP Delivery: Integrated with Gmail's secure SMTP server to automatically email encrypted files to beneficiaries.

Secure Version Control: Configured with .gitignore to prevent sensitive master keys from being exposed in public repositories.

🛠️ Tech Stack
Language: Python 3.x.

Security: AES (Advanced Encryption Standard).

Libraries: cryptography, smtplib, os, datetime.
